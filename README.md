# Bubble Bobble Clone
### Interactive Media - 4160U
In this assignment, we were asked to rebuild Wireframe Magazine's "Cavern" (pygame reconstruction of Bubble Bobble) in Godot. The original project can be found here: <a href="https://github.com/Wireframe-Magazine/Code-the-Classics/blob/master/cavern-master/cavern.py">Link</a>

## How to run
- Open the project in Godot 4.x
- Run the project with the main scene set to scenes/screens/main_menu.tscn

## Controls
Arrow Keys Left+Right = Move left/right
Arrow Key Up = Jump
Spacebar = Fire bubble

## Implemented features
- ✅ Player movement + jump
- ✅ TileMap collisions
- ✅ Bubble shooting
- ✅ Enemies patrol
- ✅ Trap mechanic
- ✅ Pop + score + HUD
- ✅ Game over + restart
